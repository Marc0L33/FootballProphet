## 双层架构

**规则层**：每条规则维护贝叶斯追踪状态。17 条规则各自独立更新。

**比赛层**：给定一场比赛，预测 = 基线 + 各触发规则的效应量贡献，冲突时按优先级仲裁。

### 预测公式

```
进球_A = ⑭基线_GF × ②对手系数                          ← 大样本基线（直接计算，不进贝叶斯）
       + Σ(触发规则_i × γ_i × δ_i)                      ← 各规则的信度加权效应量
       → 被⑤硬上限截断（如触发）

进球_B = 同上，对称计算

最终比分 = (round(进球_A), round(进球_B))
```

> ⑭ 和 ② 不进贝叶斯——它们是直接从数据计算的值（GF/场、对手级别），不是需要推断的参数。

## 贝叶斯追踪字段

每条规则维护：

```
规则 X：
  α    方向正确次数 + 1（Beta-Bernoulli 先验伪计数）
  β    方向错误次数 + 1
  E[γ] 后验信度均值 = α / (α+β)。衡量「这条规则方向判断对」的概率
  δ    效应量均值（触发后预期偏移多少球）
  σ    效应量不确定性（标准差）
  n    触发并被评估的总次数
```

## 增量更新公式

每场比赛复盘后，对每条触发规则执行：

### 信度更新（Beta-Bernoulli 共轭）

```
复盘判定：本场该规则的方向判断是否正确？
  → 正确：α_new = α_old + 1
  → 错误：β_new = β_old + 1
  → E[γ] = α_new / (α_new + β_new)
```

### 效应量更新（正态-逆伽马共轭）

```
观测到实际效应量 x（该规则本场实际造成的进球偏移）

超参数更新：
  κ_n = κ_0 + n
  μ_n = (κ_0 × μ_0 + n × x̄) / κ_n
  α_n = α_0 + n/2
  β_n = β_0 + 0.5 × Σ(x_i - x̄)² + (κ_0 × n × (x̄ - μ_0)²) / (2 × κ_n)

后验：
  δ（效应量均值）= μ_n
  σ（效应量标准差）= √(β_n / (α_n × κ_n))
```

### 更新触发条件

- 规则被显式触发且用于预测：执行上述更新
- 规则应触发但未被触发：不自动更新参数，标记为「待审查」，由人工决定是否调先验
- 更新粒度：**比赛级别**（一场比赛触发规则⑤ → 记一次 ±1，不按队分别记）

## 新规则冷启动（影子模式）

```
新增规则 → 弱先验 α=1, β=1（E[γ]=0.50, 完全不确定）
         → 状态标记：🆕 影子模式
         → 不参与预测（γ 太低，贡献 ≈ 0）
         → 每场复盘照样更新 α/β/n
         → 准入条件：E[γ] > 0.60 且 n ≥ 5 → 去掉 🆕，正式上线
```

### 冷启动流程

```
1. 复盘发现新信号 → 提案为新规则
2. 写入 rules.md，初始状态：
     状态：🆕 影子模式
     α=1, β=1, E[γ]=0.50, n=0
3. 每场复盘更新 α/β/n，但不参与预测
4. 当 E[γ] > 0.60 且 n ≥ 5 → 自动转正，去掉 🆕
5. 当 n ≥ 10 且 E[γ] < 0.50 → 规则无效，删除
```

## 既有规则退出机制

影子模式是入口——退出机制是出口。对称设计：

```
触发条件（满足任一即启动）：
  1. E[γ] < 0.52 且 n ≥ 10 → ⚠️ 降级为影子模式，暂停参与预测
  2. E[γ] < 0.50 且 n ≥ 15 → ❌ 从方法论中永久删除
  3. 连续 5 场方向错误 → 🔍 强制审查（不论 n 和 E[γ]）

降级后的处理：
  → 影子模式中仍更新 α/β/n
  → 若 E[γ] 回升至 >0.55 → 重新上线
  → 若 n ≥ 20 且 E[γ] < 0.50 → 永久删除
```

## 效应量叠加规则

1. **加性规则**（⑤⑩⑫⑬⑮⑯⑱⑧⑦③①）的 δ 直接相加。⑫⑦⑧③ 的 δ 按场计算但叠加方式不变
2. **乘性规则**（⑰②）在加性之前应用。⑰ 先校准基线 GF，② 再按对手调整
3. **校验规则**（⑪⑨）不产出效应量，不参与叠加
4. 总效应量不超过 ±2.0 球（⑫ × ⑦ 交互条款）
5. ⑤ 触发时，该队 cap=1（硬约束，不可被任何其他规则突破）

## 预测区间计算

```
总方差 = Σ(触发规则_i 的 σ_i²)
80% 后验区间 = [预期进球 - 1.28 × √总方差, 预期进球 + 1.28 × √总方差]
```
